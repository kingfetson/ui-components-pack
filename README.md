# 📁 Project Structure

Each component is organized in its own folder.

Example:

```

navbar-01-minimal/
│
├── index.html
├── style.css
├── script.js
└── preview.png

````

All navbars follow the same structure for easy editing.

---

# 🚀 How to Use

1. Open the navbar folder you want to use.

2. Copy the following files into your project:

- `index.html`
- `style.css`
- `script.js`

3. Customize:

- Logo
- Menu links
- Colors
- Fonts
- Icons

You can also merge the CSS into your main stylesheet.

---

# 🎨 Customization

### Change Logo

```html
<div class="logo">YourBrand</div>
````

### Change Colors

```css
.navbar {
background: #2563eb;
}
```

### Add Menu Links

```html
<li><a href="#">Blog</a></li>
<li><a href="#">Pricing</a></li>
```
